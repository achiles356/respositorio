Passos para Hospedar um Projeto no GitHub Pages

  • Crie um repositório no GitHub com o mesmo nome do seu projeto
  • No terminal, inicialize o repositório local com git init
  • Altere o nome da branch principal para main
  • Adicione os arquivos com git add . e faça o commit com git commit -m "first commit"
  • Conecte com o repositório remoto com:
  • git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  • No GitHub, vá em Settings > Pages.
  • Em "Branch", selecione main e a pasta / (root) . Clique em save
  • Após alguns segundos, seu site estará disponível no link exibido em "Pages"
